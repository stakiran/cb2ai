# cb2ai
クリップボード内容を OpenAI API に送信して、結果をテキストファイルに書き込んでから開く Python スクリプト。

## usage

```
$ python cb2ai.py --prompt (prompt-file) --model (model-name) --timeout (timeout-seconds)
```

なお、 -model と --timeout は未指定も可能。その場合はデフォルト値を使う。

## プロンプトファイル
- プロンプトを記載する
- 変数 cb をサポートする
    - `%cb%` と書く
    - この部分には、現在のクリップボード内容が挿入される

### 例
意図を問うプロンプト:

```markdown
%cb%

これってどういうこと？
```

言葉の意味や定義を調べるプロンプト:

```markdown
%cb%

---

上記のうち、一行目の内容のみをピックアップした上で、その概要を Markdown 形式でレポートして。
多義語の場合は、そのすべてをレポートして。
```

## AI への問い合わせ
- OpenAI API を使う
- 使用 API は Chat Completions で良い
- 使用モデルはコマンドライン引数から指定する
    - API リクエスト時に指定する文字列をそのまま指定する。たとえば GPT 4o を使いたい場合は `gpt-4o` となるはずだ
- タイムアウト秒数もコマンドライン引数から指定する
    - デフォルトは 130 秒で良い

## 問い合わせ結果の返却
- テンポラリフォルダにテキストファイルを生成し、そこに結果を書き込んだ上で、そのファイルを開く
- テンポラリフォルダは Windows 向けに %temp% で良い
- テキストファイルは Markdown であり、ファイル名は `cb2aiXXX.md` とする
    - `XXX` はランダムな文字列で良い
- ファイルオープンは、関連付けられた手段で開けばいい。ノンブロッキングで開くこと
